h1. Throbbo

Throbbo is a jQuery plugin that makes is easy to display a throbber (aka spinner) while loading.

h2. Usage

Show throbber as an overlay in *#foo*. Throbber will show for 2 sec.

<code>
$('#foo').throbbo({duration: 2000});
</code>
